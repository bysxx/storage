Section 02

통합 마케팅 대시보드

크리에이팁이 일룸(iloom) 사례를 기반으로 구성한 통합 마케팅 대시보드 목업을 시연하며 주요 기능을 설명하였다. 현재 더미 데이터로 구성된 목업 단계이며, 데스커의 요구에 맞춰 커스터마이징 가능하다.

대시보드 주요 기능

기능 설명 데이터 소스
검색 트렌드 데스커 주요 키워드에 대한 검색량, 트렌드 변화 분석 네이버·구글 등 서치엔진 API
소셜 트래킹 멘션 수, 플랫폼별 현황(인스타그램·유튜브·네이버 블로그), 도달률, Impression 브랜드워치 등 고가 API
센티먼트 분석 AI 활용 긍정/부정 반응 분석 소셜 데이터 + AI 분석
해시태그 트래킹 인스타그램·X 등 해시태그 사용량 모니터링 각 플랫폼 API
경쟁사 분석 합의된 경쟁사 선정 후 지표 비교 커스텀 설정
광고 모니터링 GA/GTM 성과 데이터, 캠페인별 노출·클릭·CTR 통합 조회 구글·메타·네이버·크리테오 API
리포트 생성 도메인 지식 없이도 쉽게 리포트 생성 가능 대시보드 내 데이터
설정 브랜드 세팅, 경쟁사 설정, 채널 연동, 팀원 관리 -

데이터 수집 방식

다층적 데이터 수집 체계

  • 서치엔진 API — 네이버, 구글 등 검색 트렌드·검색량 데이터
  • 고가 API (브랜드워치 등) — 해외 상장 기업의 공신력 있는 소셜 데이터, 메타 등 접근 제한 플랫폼도 계약 기반 제공
  • 광고 플랫폼 API — 구글, 메타, 네이버(GFA 파트너 업체), 크리테오, 틱톡 등
  • 자체 크롤러 — 네이버 스마트스토어, DC인사이드 등 외부 커뮤니티까지 크롤링 가능

모든 로우 데이터에 원본 링크를 제공하여 데이터 정확성을 직접 검증할 수 있도록 함

데스커 측 피드백 및 요구사항

광고 모니터링에서 캠페인 구분 없이 총 광고비를 통합 조회할 수 있는가?
데스커 팀: 캠페인별 구분값이 아닌 필터값으로 처리되어야 함. B2B 광고를 빼고 보거나, 여러 캠페인을 합쳐서 보는 등 유연한 조회가 필요. 담당자별로 캠페인이 다르고, 온라인 영업팀과 합쳐진 이후 캠페인을 넘나들며 데이터를 봐야 하는 니즈가 큼.

크리에이팁: 기술적으로 가능. 구글·메타·네이버·크리테오 등 여러 API 연동 상태이며, 캠페인명 세팅 약속에 따라 통합 조회 지원 예정.
소셜 트래킹에서 실제 콘텐츠로 바로 이동할 수 있는가?
데스커 팀 질의. 현재 목업에서는 클릭 불가하나, 실제 구현 시 원본 콘텐츠 링크 연결 가능.
메타(Meta) 데이터의 정확도와 범위는?
데스커 팀 질의. 메타 본사에서도 외부 데이터 제공을 제한하고 있다고 들었는데, 전체 대상 데이터인지 궁금함.

크리에이팁: 브랜드워치 등 해외 상장 기업과의 계약을 통해 데이터 확보. 로우 데이터에 원본 링크가 달려있어 실제 존재하는 데이터임을 검증 가능.
네이버 스마트스토어 매출 정보를 API로 가져와서 광고 성과와 함께 볼 수 있는가?
데스커 팀 질의. 기술적으로 가능. 스마트스토어 센터 접근 권한을 오픈해 주시면 매출 데이터, 리뷰 데이터 연동 가능. 경쟁사 리뷰·판매량 예측 비교도 가능.

데스커 팀 우선순위 정리

  • 최우선: 소셜 트래킹 (새로운 기능, 한 곳에서 검색·소셜 트렌드 통합 조회)
  • 높은 관심: 광고 모니터링 (캠페인 통합 조회, 필터 기반)
  • 제외 가능: 감성어(센티먼트) 분석 — 내부에서 별도 서비스 운영 중
  • 추가 검토: 스마트스토어 매출 연동, 외부 커뮤니티(DC인사이드 등) 크롤링
Section 03

GEO (Generative Engine Optimization)

크리에이팁이 GEO의 개념과 서비스 범위를 설명하였다. SEO의 진화 개념으로, AI 검색 엔진(ChatGPT, Perplexity, Google AI Overview 등)에서의 브랜드 노출을 최적화하는 작업이다.

GEO 핵심 개념

GEO란?

AI 검색 엔진(ChatGPT, Perplexity, Google AI Overview 등)이 콘텐츠를 읽고 사용자에게 추천할 때, 우리 브랜드의 제품이 우선적으로 노출되도록 최적화하는 작업. 예를 들어 "모션 데스크 추천해줘"라고 AI에 질문했을 때 데스커 제품이 가장 먼저 추천되도록 유도한다.

GEO 핵심 요소

요소 설명
링크빌딩 공신력 있는 매체(중앙일보 등)에 데스커 URL이 포함된 기사·콘텐츠 게재. AI가 높은 점수를 부여하는 핵심 요소
메타 태그 최적화 웹사이트 내 메타 태그, description 등의 점수 판별 및 개선 (일룸의 경우 메타 태그 중복 이슈 발견)
상세페이지 개선 제품 상세페이지에 FAQ 등을 녹여 AI가 잘 읽을 수 있도록 구조화
콘텐츠 발행 GEO 노출에 최적화된 콘텐츠 제작 및 발행 (내부 공식몰 or 외부 매체)
KPI 측정 AI 검색 시 실제 노출도를 직접 측정, 키워드별 노출률 추적

AI가 참고하는 공신력 있는 출처

뉴스 매체
중앙일보 등 주요 언론
커뮤니티
Reddit 등 활성 커뮤니티
유튜브
캡션 데이터 참고
공식 사이트
브랜드 공식 웹페이지

공신력 있는 출처 리스트는 AI 엔진별로 어느 정도 정해져 있으며, 해당 가이드를 제공할 수 있다고 설명. Reddit의 경우 한국에서는 사용률이 낮지만, AI 엔진에서 높은 점수를 부여하고 있어 영향력이 크다고 언급.

GEO 대시보드 연동

현재 목업에는 GEO 기능이 포함되어 있지 않으나, 대시보드에 GEO 모니터링 기능 탑재가 가능하다. 플랫폼별 키워드 검색량, 노출률 등을 대시보드에서 라이브로 확인할 수 있도록 구성할 수 있다.

데스커 측 GEO 입장

  • GEO는 진행 예정이나, 여러 업체를 탐색 중인 단계
  • 수치 확인 + 컨설팅까지 가능한 업체를 비교 검토 중
  • 당장 결정은 어려우며, 진행 희망 시 추가 논의 예정

Section 05

주요 Q&A

네이버 GFA API도 접근 가능한가?
네이버 GFA API는 파트너 업체에만 공개되어 있으나, 크리에이팁이 파트너 업체이므로 데이터를 받아 전달 가능.
대시보드를 도입하면 코디네이터 팀의 업무가 줄어드는가?
데스커 팀 질의. 광고 현황·성과 관련 업무 리소스가 줄어들 수 있으며, 데스커의 고도화된 데이터를 크리에이팁도 활용하고 싶어 이 프로젝트를 고무적으로 보고 있음.
외부 커뮤니티(DC인사이드 등)도 소셜 트래킹에 포함 가능한가?
데스커 팀 질의. 가능하나 API가 아닌 크롤링 방식이 되며, 시간과 비용이 추가로 소요됨.
GEO에서 출처별 참고 비율(퍼센테이지)을 확인할 수 있는가?
정확한 확인이 필요하나 아마 가능할 것으로 보임. 공신력 있는 출처 리스트는 어느 정도 정해져 있어 가이드 제공 가능.
상세페이지 개선을 크리에이팁이 직접 개발할 수 있는가?
기술적으로 가능하나, 퍼시스 내부 보안 규정상 소스 오픈이 어려울 수 있음. 3가지 방향 제시:
① 웹사이트 구조 자체를 개발적으로 변경 (개발팀 투입)
② 상세페이지별 SEO 요소를 개별 수정
③ 컨설팅 방향으로 가이드 제공 (개발 권한 부여 불가 시)
Reddit에 콘텐츠를 올리면 AI가 공신력 있게 여기는가?
한국에서 Reddit 사용률이 낮아 바이럴 느낌이긴 하지만, AI 엔진에서 꽤 높은 점수를 부여하고 있어 영향력은 큼. 여타 커뮤니티도 팔로우업 필요.

다음 주까지 (코디네이터)

견적 재산정 후 기능별 버전으로 메일 정리하여 전달. 소셜 트래킹 포함/미포함 등 옵션별 견적 제시 예정. 일룸보다 경쟁력 있는 금액으로 조율 목표.

이후 (데스커 팀)

견적 검토 후 내부 논의 및 상부 보고. 필요한 기능 범위를 좁혀 요구사항 전달.

진행 결정 시 (양 팀)

수차례 회의를 통해 기능 확정 → 견적 픽스 → 본격 개발 착수

추가 참고 사항

  • 크리에이팁과 GEO를 진행하지 않더라도, GEO 관련 지식이나 개발 관련 Q&A는 언제든 문의 가능 (코디네이터 경유)
  • 데이터 커스터마이징 요청도 가능하며, 필요한 데이터 종류를 말씀해 주시면 맞춤 대응
  • 대시보드는 일룸 요청 기반으로 만든 목업이므로, 데스커 요구에 맞춰 기능 가감 가능
